RescueSession Class Reference

Inherits from RescueSingleton : NSObject
Declared in RescueSession.h

Overview

The connection between your application and the Rescue Technician Console is represented by a RescueSession object. This is a singleton object you access it by calling the sharedInstance class method.

Identification

  apiKey

Unique identifier for the app.

@property (nonatomic, strong) NSString *apiKey

Discussion

Every app need an API key, without it starting a Rescue session will always fail.
