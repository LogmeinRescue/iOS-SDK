RescueSession Class Reference

Inherits from RescueSingleton : NSObject
Declared in RescueSession.h

Overview

The connection between your application and the Rescue Technician Console is represented by a RescueSession object. This is a singleton object you access it by calling the sharedInstance class method.

addLogger:withLevel:

Subscribe to get log messages from Rescue SDK.

- (void)addLogger:(id<RescueLoggerDelegate>)loggerDelegate withLevel:(RSLogLevel)level

Parameters

loggerDelegate

The object who will receive log messages.

level

The receiver will get messages on this or above this level. See RSLogLevel in RescueLoggerDelegate.h.

Discussion

Async call, see delegate methods in RescueLoggerDelegate to get feedback about the successfulness of subscription.
