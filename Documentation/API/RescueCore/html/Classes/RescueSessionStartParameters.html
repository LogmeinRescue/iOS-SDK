Inherits from NSObject
Declared in RescueSessionStartParameters.h

Overview

The RescueSessionStartParameters class is used by the RescueSession class to configure the session start parameters. Do not create your own instance of this class, RescueSession will hold its own, use that one.

Instance Methods

continueSession

Configure the session start parameters to try to continue the last unfinished session.

- (BOOL)continueSession

Return Value

YES if there is an unfinished session, session start parameters is configured to continue this session or NO if there is no session to continue, session won’t start with this configuration.

Declared In

RescueSessionStartParameters.h

isConfigured

Returns a boolean indicating weather the instance holds a valid configuration to start a session.

- (BOOL)isConfigured

Return Value

YES if configuration is valid to start a session, otherwise NO

Discussion

Returning YES this method does not guarantee, that the session will start, or the given parameters are valid. It just checks for the existence of the parameters.
