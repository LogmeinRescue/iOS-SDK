Overview

The RescueSessionDelegate protocol defines a set of methods you can use to get notified about session related events.

rescueSessionError:withUserInfo:

Tells the delegate that an error occured during session.

- (void)rescueSessionError:(RescueError)errorCode withUserInfo:(NSDictionary *)userInfo

Parameters

The error code.

userInfo

More info about the error. In case of RescueErrorNoTechnicianAvailable, the dictionary contains the URL set in the administration center if there are no technicians on the channel.
