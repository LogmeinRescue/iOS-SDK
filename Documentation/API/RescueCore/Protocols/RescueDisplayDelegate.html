RescueDisplayDelegate

@objc
public protocol RescueDisplayDelegate

The RescueDisplayDelegate protocol defines a set of optional methods you can use to get notified about display streaming related events.

All of the methods in this protocol are optional.

  • rescueDisplayRequestViewToSendToTechnician()

    Ask the delegate for a view object to use as root view for display streaming.

    This method is called if RescueDisplay‘s mode is set to RootView but no root view was provided.

    Declaration

    Swift

    @objc

    Return Value

    A view object from the view hierarchy to use as root view for display streaming.
