RescueSessionDelegate

The RescueSessionDelegate protocol defines a set of methods you can use to get notified about session related events.

  • rescueSessionStatusDidChange(_:)

    Tells the delegate that the state of the session has changed.

    This method is optional.

  • rescueSessionError(_:withUserInfo:)

    Tells the delegate that an error occured during session.

  • rescueSessionReady()

    Notify the delegate that the session is ready for connecting or broadcasting
