RescueSession

public class RescueSession : NSObject

The connection between your application and the Rescue Technician Console is represented by a RescueSession object. This is a singleton object you access it by calling the sharedInstance class method.

reachability

  • shouldUseInAppBroadcastPicker()

    In certain iOS versions RPSystemBroadcastPickerView can be embedded to the app to start broadcast session easier.

    RPSystemBroadcastPickerView has known issues:

    • on iOS 12 before 12.2 the preferredExtensionfunction does not work properly
    • on iOS 13 before 13.1 app crashes when user presses button

    The function checks if the picker can be used on the iOS version running the app.

    Declaration

    Swift

    @objc
public class func shouldUseInAppBroadcastPicker() -> Bool

    Return Value

    true if RPSystemBroadcastPickerView is available and works, otherwise false.
