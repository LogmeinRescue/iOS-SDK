RescueSession

public class RescueSession: NSObject, RescueBackgroundDelegate, RescueGatewayDelegate, RescueSessionCreatorDelegate, RescueChatManagerDelegate, RescueChatMessagePacketHandlerDelegate, RescueDeviceViewManagerDelegate

The connection between your application and the Rescue Technician Console is represented by a RescueSession object. This is a singleton object you access it by calling the sharedInstance class method.

Control session lifecycle
  • startSession()

    Start a Rescue session using the configuration from the sessionStartParameters object.
