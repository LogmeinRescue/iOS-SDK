RescueDisplayDelegate Protocol Reference

Conforms to NSObject
Declared in RescueDisplayDelegate.h

Overview

The RescueDisplayDelegate protocol defines a set of optional methods you can use to get notified about display streaming related events. All of the methods in this protocol are optional.

Instance Methods

rescueDisplayRequestViewToSendToTechnician

Ask the delegate for a view object to use as root view for display streaming.

- (UIView *)rescueDisplayRequestViewToSendToTechnician

Return Value

A view object from the view hierarchy to use as root view for display streaming.

Discussion

This method is called if RescueDisplay’s mode is set to RescueDisplayModeRootView but no root view was provided.
